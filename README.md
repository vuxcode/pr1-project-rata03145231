# Project Summary

Project Presentation
The project is a simple browser game based on Rock, Paper, Scissors. The player plays against the computer, and the game keeps track of the score. HTML and CSS are used for structure and design, and JavaScript is used to make the game work.
Reflection
The biggest problem I had was understanding JavaScript. Compared to HTML and CSS, JavaScript was much harder to understand. With HTML and CSS, it felt easier to see what I was doing and create something from nothing. Because JavaScript felt more difficult, I chose to make a simpler project.
The biggest struggle in the code was the reset button. I had trouble understanding how to reset the game and the scores. I eventually solved it in a way similar to a YouTube video I had seen, but it took time to understand how it worked.
A smaller struggle was handling the score for the player and the computer. Once I realized I could use an if statement for a draw, an else if for player wins, and an else for losses, the logic became much clearer.

Starting the Game

When the page loads, the game starts automatically. No setup is required. The score begins at Player 0 | Computer 0.

How to Play

Choose one option by clicking a button (for example Rock, Paper, Scissors, Gun, Shield, Bomb, Lawyer, Criminal, or Cop).

The computer will automatically choose a random option.

The result of the round will be shown on the screen, including:

What you chose

What the computer chose

Who won the round

Scoring

If you win a round, your score increases by 1.

If the computer wins a round, the computerâ€™s score increases by 1.

If both choose the same option, the round is a draw and no points are given.

Winning the Game

The first to reach 3 points wins the game.

A message will appear announcing the winner.

The game will automatically reset after a win.

Reset Button

Clicking the Reset button will reset the game at any time.

Both scores will return to 0 and the round result text will be cleared.

Ending the Game

To stop playing, simply close or reload the browser page.
